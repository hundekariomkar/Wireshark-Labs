# Wireshark-Labs
📡 Wireshark Network Traffic Analysis Project 🦈

A structured network traffic analysis project leveraging Wireshark to inspect core internet protocols and identify potential security threats at the packet level.

This repository demonstrates hands-on expertise in protocol analysis, anomaly detection, and foundational network forensics.

---

📌 Project Overview

Modern networks face increasing risks from misconfigurations, reconnaissance scans, denial-of-service attempts, and data interception.

This project focuses on:

Deep packet inspection

Protocol behavior analysis

Identification of suspicious traffic patterns

Practical use of Wireshark for security investigations

---

🎯 Objectives

Analyze real network traffic using Wireshark

Understand HTTP, HTTPS, DNS, TCP, and ICMP behavior

Apply protocol-specific filters

Detect anomalies and suspicious patterns

Document findings in a structured security format

---

| Tool                    | Purpose                              |
| ----------------------- | ------------------------------------ |
| 🦈 Wireshark            | Packet capture & protocol inspection |
| 🖥 Kali Linux / Windows | Analysis environment                 |
| 🌐 Web Browser          | Generate HTTP/HTTPS traffic          |
| 📡 ping / nslookup      | Generate ICMP & DNS traffic          |

---

📡 Topics Covered

🌐 HTTP Traffic Analysis

Request/Response inspection

Header analysis

Clear-text payload visibility

🔐 HTTPS Traffic Analysis

TLS handshake observation

Certificate exchange

Encrypted session behavior

📡 DNS Query & Response Investigation

A/AAAA/MX record inspection

Query frequency analysis

Suspicious domain detection

🧠 TCP 3-Way Handshake & Flags

SYN, SYN-ACK, ACK sequence

Retransmissions

Window size & connection termination

📶 ICMP Network Diagnostics

Echo request/reply

TTL analysis

Error message interpretation

🚨 Suspicious Traffic Detection

Abnormal SYN patterns

DNS anomalies

Traffic spikes

Protocol misuse indicators

---

🔬 Methodology

Generate controlled network traffic.

Capture packets using Wireshark.

Apply display filters for protocol isolation.

Inspect packet headers and payload structure.

Identify normal vs abnormal behavior.

Document findings and security implications.

---

📊 Results

Successfully analyzed multiple protocols at packet level

Identified normal communication patterns

Detected potential suspicious behaviors

Demonstrated structured investigation workflow

Strengthened understanding of network security fundamentals

⚠ Limitations

Limited to controlled lab environment

Does not simulate enterprise-scale traffic
